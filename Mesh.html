In this section, we present some of the mesh definition and manipulation tools provided by Feel++.
Feel supports only the Gmsh mesh file format. It provides also some classes to manipulate Gmsh .geo files and generate .msh files.

The Gmsh class helps with the generation of meshes using the program gmsh.

You can visualize the geometry files .geo and .msh using the gmsh application : 

  gmsh <entity_name>.msh
