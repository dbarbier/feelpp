Detailed Description

template<typename Poly, template< uint16_type > class PolySetType = Scalar>
class Feel::PolynomialSet< Poly, PolySetType >

a Set of polynomials

This class represents a set of polynomials defined in a certain basis given by the template argument. The coefficients of the polynomials in the basis are represented by a matrix whose line represents the polynomials and columns the basis index where is the mapping between the polynomial and its coefficients.
